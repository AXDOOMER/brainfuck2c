# brainfuck2c

Brainfuck to C transcompiler written in Bash.

## Requirements

Have `bash`, `sed`, `which` and `gcc` installed. Most Linux distributions have them already installed.

## Usage

`./trans.sh foobar.bf`

On success, this creates `foobar.bf.bin`.
